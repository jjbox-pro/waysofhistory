Создание плагина:

1) npm i -g plugman (если не установлено)

2) plugman create --name CordovaCustomPlugin --plugin_id cordova-plugin-name --plugin_version 1.0.0

3) plugman platform add --platform_name android
	
4) plugman createpackagejson ./
